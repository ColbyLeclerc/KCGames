=== Folder Structure ===

Scripts: Contains the .sql scripts to create each table and populate them. This folder also contains a file named 'backup.sql' which is the entire database exported to a single file.

Interface: Contains the folder for each of our websites. This includes the images for the site, as well as the .php files to run each of the queries.

Diagrams: The ERD and Schema image files

Reports: The final report and proposal

Queries: The SQL file for each query used in the final project
